# ARC Studio Remake 

An all-in-one, open-source studio for editing Minecraft Legacy (PS3) ARC files.

## Features

*   **User-Friendly UI:** Intuitive interface designed for both beginners and advanced users.
*   **All-in-One Workflow:** Edit both ARC archives and LOC languages.loc files in a single application.
*   **Robust & Safe:** Built-in backup options and validation to prevent save corruption.
*   **Optimized Performance:** Handles large game files efficiently.

## System Requirements

* Windows 10/11
* x64/x32 Processor

## Download

Head to the [Releases](https://github.com/DescZDev/ARC_Studio_Remake/releases) page to download the latest version of ARC Studio.

## Usage

1.  Extract your Minecraft PS3 game file.
2.  Open ARC Studio on your desktop.
3.  Drag and drop your `arc` file into the window.
4.  Make your edits and save!
